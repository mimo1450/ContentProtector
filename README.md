# ContentProtector
How to use?

add the following to the end of your html page's body
<script>
    var noPrint=true;
    var noCopy=true;
    var noScreenshot=true;
    var autoBlur=true;
</script>
<script type="text/javascript" src="protector.js"></script>

you can turn options on/off as needed

enjoy it :)
