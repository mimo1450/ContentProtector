# ContentProtector
How to use?

add the following to the end of your html page's body
<script><br/>
    var noPrint=true;<br/>
    var noCopy=true;<br/>
    var noScreenshot=true;<br/>
    var autoBlur=true;<br/>
</script><br/>
<script type="text/javascript" src="protector.js"></script><br/>
<br/>
you can turn options on/off as needed
<br/>
enjoy it :)
